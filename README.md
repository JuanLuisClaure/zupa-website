## Getting Started

git pull
npm install or yarn install
npm start or yarn start

it will create a www folder with production code

build/ folder is for client side
src/client/ folder is bundle build code
src/server/ folder is for server side

### Prerequisites

I think just gulp global install and better if you use Yarn global too
